# Duplicate Question Detection using Deep Learning

## 📌 Description
This project detects duplicate questions on Stack Overflow using NLP and deep learning techniques.

## 🚀 Features
- Text preprocessing using NLTK
- Duplicate detection model
- Dataset-based evaluation

## 🛠️ Technologies Used
- Python
- NLTK
- Machine Learning / Deep Learning

## 📂 Project Structure
- src/ → source code
- data/ → datasets
- docs/ → project documents

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python src/DuplicateQuestionDetection.py

## 📊 Dataset
Dataset contains question pairs labeled as duplicate or not.
